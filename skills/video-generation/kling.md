# Kling

**Best for:** realistic motion quality and high-detail action scenes.

## Public Model Signals

Official Kling channels currently highlight the Kling 2.x family.
Recent ecosystem references include 2.1 and 2.6 variants.

## API Access Reality

- Public product pages expose model updates clearly
- Direct official API documentation is less discoverable in open web indexing
- Production teams frequently use partner or unified API gateways

## Known API Surface in Ecosystem

- Tencent MPS lists Kling model options (`kling`, `kling2_1master`, `kling2_6`)
- Partner-hosted APIs expose Kling variants for programmatic use

## Practical Routing

- Use Kling for human motion, action, and realistic movement dynamics
- Keep prompts physically plausible to prevent temporal artifacts
- Use short test clips before generating longer sequences

## Reliability Tips

- Include explicit motion pacing in prompts
- Keep character wardrobe and scene cues stable across cuts
- Cache final outputs immediately after completion
