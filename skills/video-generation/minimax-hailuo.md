# MiniMax Hailuo

**Best for:** expressive motion scenes and rapid iteration in partner APIs.

## API Access Reality

- MiniMax ecosystem models are widely available through hosted providers
- Fal documents current Hailuo API model families
- Tencent MPS gateway includes `hailuo-02` and `hailuo-2.3`

## Practical Routing

- Use Hailuo for dynamic character motion and energetic scene timing
- Keep scene instructions direct and avoid over-complex camera stacks
- Validate with short drafts before longer renders

## Reliability Tips

- Separate subject/action/camera lines in prompts
- Track provider-specific limits and queue latency
- Keep one premium fallback model for final rerenders
