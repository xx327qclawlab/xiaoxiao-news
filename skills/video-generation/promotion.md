# AI Video Promotion Playbook

Use this after generation quality is already validated.

## Channel Fit

- YouTube Shorts, Reels, TikTok: generate native `9:16`
- YouTube long form: generate `16:9` and repurpose clips for shorts
- LinkedIn: keep concise, subtitle-first explainers

## High-Impact Iteration Loop

1. Generate 3 to 5 hook variants
2. Keep only top retention hooks
3. Rerender winners with premium model tier
4. Add subtitles and strong CTA

## What to Track

- Hook retention at 3 seconds
- Completion rate by duration
- CTR to landing page
- Cost per qualified view

## Common Distribution Mistakes

- Publishing horizontal clips on vertical-first platforms
- Testing only one hook per creative idea
- Over-optimizing visuals before message validation
- Ignoring subtitles on muted autoplay feeds
